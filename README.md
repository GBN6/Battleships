# Battleships
Battleship game versus computer. Built using TDD(Test Driven Development)

# LIVE VERSION

https://gbn6.github.io/Battleships/


### Features

* Change axis button
* Hover to place ships
* Restart after win
* Responsive design.

### Built With

#### Technologies

* JavaScript
* CSS
* HTML

#### Third party code

* [Google Fonts](https://fonts.google.com/)

#### Tools

* Visual Studio Code
* Linux terminal
* Git and GitHub
* Webpack
* Jest
* ESLint + Airbnb JavaScript Style Guide
* Prettier
